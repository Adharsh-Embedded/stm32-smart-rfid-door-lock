# STM32 Smart RFID Door Lock

## Components Used
- STM32F446RE
- RC522 RFID Module
- HC05 Bluetooth Module
- Servo Motor
- I2C LCD

## Features
- RFID authentication
- Servo-based door lock
- Bluetooth notification
- LCD status display

## Communication Protocols Used
- SPI
- UART
- I2C
- PWM
